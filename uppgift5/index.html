HTML

HTML är en förkorting och står för "HyperText Markup Language". HTML är ett språk som används för att göra webbsidor. Från början skapades HTML för att stadardisera sätt att skriva vetenskapliga dokument och för att kunna läsa av dem skapade han en webbläsare.

HTML vs CSS

Den stora skillnaden mellan HTML och CSS är att:

HTML används för strukturera upp ett dokument.

CSS används för att ändra utseendet på dokument som är skrivna i HTML.

CSS i HTML

När man vill använda CSS i html finns det två huvudskaliga sätt att koppla det.

Externt

Det externa sättet används när designen ska appliceras till flera sidor och inte bara en. Då länkar man till ett annat dokument som innehåller information om hur sidan ska se ut.

Internt

Om du använder det interna sättet så skriver du det direkt i HTML texten och inte i ett separat dokument. Detta är mest användbart om det är bara ett dokument som ska ändras.

Viktiga html-element

Head

  • DOCTYPE
    • Talar om för webbläsaren vilken slags dokumenttyp det är.
  • HTML
    • Vissar att följande information är i HTML.
  • HEAD
    • Head innhåller viktig information för webbläsaren.
  • TITLE
    • I title skriver du ett passade namn på ditt dokument som sedan kommer att synas på fliken i webbläsaren.

Sektioner

  • BODY
    • I BODY hittar du den största delen av ett HTML dokument, bland annat text och bild.
  • ARTICLE
    • Article representerar en oberoende del av en sida.
  • HEADER
    • Header används för att föra en rubrik.
  • NAV
    • Nav hjälper till med navigation och gör så man får ett överseende för dokumentet. Mer än att länka till delar av dokumentet så kan du även länka till andra dokument.
  • SECTION
    • Grupperar en del av sidan.
  • FOOTER
    • Hamnar i botten av dokumentet och brukar innehålla länkar och copyright data.

Grupper

  • P
    • Paragraf och här skriver du text.
  • OL
    • Ordered list och ger dig en lista med siffror.
  • UL
    • Unordered list lista där ordningen är obetydande.
  • LI
    • En ny punkt i listan. Skrivs i ol eller ul.
  • DIV
    • Grupperar sidan. Användbart om CSS ska användas.

Element på textnivå

  • A
    • Representerar en hyperlänk som kan bestå av text eller bild.
  • EM & STRONG
    • Används för att förstärka något viktigt i texten.
  • BR
    • Byter rad. Line Break på engenlska.

HTML-attribut

Attribut skrivs efter ett element och hjälper till att mer exakt definiera elementets funktion.

  • src - anger sökvägen till en bild
  • alt - anger alternativa innehållet för en bild om den inte kan visas. Inte minst viktig för talsyntes som synskadade använder.
  • href - specificerar en sidas address. Om href attributen inte finns med blir a elementets innehåll inte till en hyperlänk.
  • lang - specificerar vilket språk elementet innehåller.

Krånglig terminologi

Vi har valt följande fem ord:

  • Element - Kommando för innehåll
  • Attribut - Definerar elemänt mer exakt.
  • Meta - Vi tror den ändrar texten och utformningen på sidan.
  • Script - Flera kommandon kör i följd.
  • Markup - Avancerat sätt att skriva på.
